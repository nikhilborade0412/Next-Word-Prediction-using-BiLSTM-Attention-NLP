# 📚 Language Modeling with WikiText-2 Dataset

## 🚀 Project Overview
This project focuses on building and analyzing a **language model** using the **WikiText-2 dataset**. The goal is to understand how models learn patterns in text data and generate meaningful sequences.

The project is implemented using Jupyter Notebooks and includes data preprocessing, model building, training, and evaluation.

## 📊 Dataset Description

The **WikiText-2 dataset** is a collection of high-quality Wikipedia articles used for training language models.

- Clean and structured text
- Suitable for NLP tasks like:
  - Language Modeling
  - Text Generation
  - Sequence Prediction
